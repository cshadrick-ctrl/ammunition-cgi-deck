# Ammunition · Automotive CGI

A presentation deck built for Vercel deployment.

## Deploy to Vercel

1. Push this folder to a GitHub repository
2. Import the repo at [vercel.com/new](https://vercel.com/new)
3. Vercel auto-detects Next.js — click **Deploy**

## Run locally

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Editing the deck

The deck lives in `public/deck.html` — it's a self-contained HTML file with all assets inlined.
To update slides, edit this file and redeploy.
